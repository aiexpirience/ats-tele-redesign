## Решения по дизайну

### Цветовая палитра
- Фон: `#f8f7f3` (тёплый белый)
- Акцент: `#1565C0` (синий)

- CTA: форма заявки прямо в hero

## Технические решения

### Архитектура темы (важно!)
Всё глобальное — **только в одном месте**:
- `header.php` — навигация
- `footer.php` — footer, Telegram кнопка, Яндекс.Метрика, глобальный JS
- Никогда не дублировать эти блоки в отдельных страницах

### AJAX форма
```php
// functions.php
add_action('wp_ajax_atctelecom_form', 'atctelecom_form_handler');
add_action('wp_ajax_nopriv_atctelecom_form', 'atctelecom_form_handler');
```
- ID формы: `reqForm`
- ID успеха: `reqSuccess`
- Поддержка прикрепления файла (фото повреждения)

### Scroll-reveal анимации
В `footer.php` есть универсальный IntersectionObserver, который автоматически применяется к `.card`, `.grid-2>*`, `.grid-3>*`, `.section-hd` на всех внутренних страницах. Пропускает главную (определяет по наличию `.hero-h1`).

### CSS переменные — единый стандарт
Все страницы используют одинаковые переменные:
```
--bg, --bg2, --card, --card2
--text, --text2, --text3
--accent, --border, --shadow, --shadow-lg
--radius, --max-w, --nav-h
```
Старые страницы (remont-apc-*) имеют алиасы для совместимости:
```css
--surface: var(--bg2);
--surface2: var(--card2);
```

## Лог изменений

### Апрель 2026 — Сессия 2 (фикс багов и унификация)

**Исправлено:**

1. **Двойная навигация** на page-contacts, page-rent, page-services — удалены дублирующие `<nav>` блоки из тела страниц (header.php рендерит nav)

2. **Двойной footer** на тех же страницах — удалены inline `<footer>` блоки, оставлен только `get_footer()`

3. **Двойной DOCTYPE/head** на page-contacts, page-rent, page-services — удалены случайно попавшие `<!DOCTYPE html>...<head>` блоки. Schema.org JSON-LD сохранён.

4. **Символ `">` на странице контактов** — причина: broken SVG в дублирующем блоке попадал в body. Решение: удалён весь блок дублирующего head.

5. **Несколько значков Telegram** — float-кнопки были в каждой странице и в footer.php одновременно. Удалены из всех страниц, оставлена только одна в footer.php.

6. **Telegram кнопка в левом нижнем углу** — причина: footer.php рендерит `<div class="float-wrap">`, но CSS `.float-wrap { position:fixed }` был только в некоторых страницах. На страницах с `.float-contacts` CSS кнопка теряла позиционирование и падала в поток. Решение: CSS для `.float-wrap` добавлен прямо в footer.php.

7. **Тёмный нечитаемый текст "АТС ТЕЛЕКОМ" в footer** — добавлен `.footer-brand { color: #fff !important }` в footer.php.

8. **Расписание на странице контактов** — исправлено: было `Пн–Пт / 9:00–18:00`, стало `Пн–Сб / 10:00–20:00` и `Вс / выходной`.

9. **"Гадкий голубой" в тёмной теме** — цвета тёмной темы были navy-blue (`#0c0e1a`, `#121520`, `#171b2a`). Заменены на нейтральный тёмно-серый (`#121212`, `#1a1a1a`, `#202020`) во всех 24 файлах.

**Добавлено:**

10. **Яндекс.Метрика** (ID: 108341486) — в footer.php с webvisor + clickmap + ecommerce. Удалены placeholder-комментарии из всех страниц.

11. **Яндекс.Вебмастер** — мета-тег `7d930fea3e86b9ce` в header.php.

12. **Реальный номер телефона** — `+7 (904) 649-09-09` заменил заглушку `+7 (812) 123-45-67` во всех 32 файлах (79 замен).

13. **Scroll-reveal анимации** — универсальный IntersectionObserver в footer.php, работает на всех внутренних страницах автоматически.

14. **Единые CSS переменные** — все страницы приведены к единому стандарту.
