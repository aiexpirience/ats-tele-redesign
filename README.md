# АТС-Телеком — Редизайн сайта ats-tele.com

> Подробный лог всех решений, правок и договорённостей: **[PROJECT-LOG.md](PROJECT-LOG.md)**

## Цели проекта

1. **Ремонт ИБП** — максимум заявок на ремонт (приоритет #1)
2. **Продажа оборудования** — конвертировать склад ~$500k в продажи
3. **Доверие** — выглядеть солиднее конкурентов, B2B аудитория СПб

## Стек

- CMS: WordPress (хостинг Timeweb)
- Репозиторий: GitHub (aiexpirience/ats-tele-redesign)

## Структура репо

```
research/     # анализ конкурентов, ЦА, боли клиентов, текущие метрики
copy/         # тексты, заголовки, офферы для каждой страницы
design/       # макеты, референсы, цветовая палитра, шрифты
wordpress/    # кастомный CSS, JS, экспорт настроек, сниппеты
```
