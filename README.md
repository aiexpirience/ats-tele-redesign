# АТС-Телеком — Редизайн сайта ats-tele.com

> Детальный лог всех решений и договорённостей: **[PROJECT-LOG.md](PROJECT-LOG.md)**

## О проекте

**АТС ТЕЛЕКОМ** — сервисный центр ИБП в Санкт-Петербурге, работает с 2009 года (17 лет), B2B клиенты.

- Сайт: https://ats-tele.com
- Хостинг: Timeweb
- CMS: WordPress, кастомная тема (без Elementor)

## Цели

1. Получать заявки на **ремонт ИБП** с сайта — приоритет #1
2. **Продавать** оборудование со склада (~$500k)
3. Выглядеть профессионально для **B2B** аудитории

## Ключевые технические решения

### Единая архитектура (без дублирования)
- `header.php` — единственное место для `<nav>`
- `footer.php` — единственное место для `<footer>`, Telegram float-кнопки, Яндекс.Метрики и всего глобального JS
- **Никогда не дублировать** эти элементы в отдельных страницах

### JS в footer.php
```
toggleMenu()      — мобильное меню
submitForm()      — AJAX форма заявки
handleFile()      — прикрепление файла к форме
toggleTheme()     — переключатель тёмной темы
applyTheme()      — применяет data-theme на <html>
IntersectionObserver — scroll-reveal на внутренних страницах
```

### CSS переменные (единый стандарт)
```css
:root {
  --bg: #f8f7f3;      /* основной фон страницы */
  --bg2: #fff;        /* вторичный фон */
  --card: #fff;       /* фон карточек */
  --card2: #f4f3ef;   /* вторичный фон карточек */
  --text: #1a1a2e;
  --text2: #4a4a6a;
  --text3: #8888aa;
  --accent: #1565C0;
  --max-w: 1240px;
  --radius: 12px;
  --nav-h: 72px;
}
[data-theme="dark"] {
  --bg: #121212;
  --bg2: #1a1a1a;
  --card: #202020;
  --card2: #282828;
  /* ... */
}
```

### Тёмная тема
Переключатель в footer.php автоматически подхватывает системную тему (`prefers-color-scheme`). Сохраняется в `localStorage`. Цвета тёмной темы — нейтральный тёмно-серый (не синий).

### AJAX форма заявки
- ID формы: `reqForm` / `reqSuccess`
- WordPress action: `atctelecom_form` (в functions.php)
- Поддерживает прикрепление файла (фото повреждения ИБП)

### Hero фото
- Файл: `hero-bg.jpg` — положить в корень папки темы
- Подключение: `get_template_directory_uri() . '/hero-bg.jpg'`
- Нужно: реальное фото склада ИБП (не серверная комната)
- Градиент: не темнее `rgba(10,15,28,.85)`

## Деплой на Timeweb

1. Зайти в FTP (данные в панели управления Timeweb)
2. Перейти в `public_html/wp-content/themes/`
3. Загрузить всю папку темы
4. В админке WordPress: Внешний вид → Темы → активировать тему
5. Проверить permalinks: Настройки → Постоянные ссылки → Сохранить
